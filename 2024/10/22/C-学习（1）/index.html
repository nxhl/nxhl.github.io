C#的输入

Console.ReadLine() 从控制台读取一行数据，返回一个字符串。

String str = Console.ReadLine();

Console.ReadKey（）检测是否按键，只要按了键盘上的任意键，就会认为输入结束。

C#的输出

Console.WriteLine（）在控制台打印信息结束后自动空一行；

Console.Write（）在控制台打印信息，但在打印完成后不会空一行。

折叠代码

由 #region 和 #endregion配对出现，可以将中间包裹的代码折叠起来，避免代码过于冗长。

#region MyRegion

#endregion

MyRegion表示要折叠的内容。

小数类变量注意事项

float存储小数时要在小数后面加一个f，加f时因为在C#中，申明的小数会默认为是double类型的，加f可告诉系统它是float系统。

decimal存储27—28位有效数字，需要在小数后面加m，一般不建议使用。

无符号的整形变量

只能存储0和正数的变量类型，不能存储负数。无符号变量可隐式转化为有符号变量，但有符号变量不可隐式转化为无符号变量。

如：byte,uint,ushort,ulong.

显式转换

括号强转：将高精度类型强制转换为低精度。语法：变量类型 变量名 = （变量类型）变量； 缺点：可能会出现范围问题造成异常。

parse法：把字符串类型转换为对应的类型（注：必须转化成对应的类型，不然会报错）。语法：变量类型.prase（”字符串”）；

Convert法：更准确的将各个类型之间进行相互转换（精度更好，会四舍五入）。语法：Convert.To目标类型（变量或常量）；

其它类型转string：可拼接打印。语法：变量.toString（）;

字符串拼接

可用+号直接拼接，如：str = str + “123”；

用+=拼接，如：当str = ”123“时，str += “1” + 4 + true（结果为 12314true）；str += 1 + 2 + 3（结果为1236）；str += ” “ + 1 + 2 + 3（结果为123123）；str += 1 + 2 + ” “ + 1 + 2（结果为123312）；

string.Format（”待拼接的内容“，内容1，内容2，……)；固定规则为：想要被拼接的内容用占位符替代{数字}中的数字，从0~n依次往后。例：str = string.Format（”我是{0}，我在{1}，我要{2}“，1，”B“，true}；

控制台打印拼接，与string.Format类似，例：Console.WriteLine（”我是{0}，我在{1}，我要{2}“，1，”B“，true）；
