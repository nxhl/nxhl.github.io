枚举

枚举（Enumeration，简称enum）是一种特殊的值类型，它允许你为一组相关的常量定义一个名称。枚举是一种强类型的常量集合，这意味着每个枚举成员都有一个在枚举类型中唯一的名称，但它们都有相同的底层类型（通常是整数类型，如int）。

声明枚举语法如下：

enum EnumName
{
Member1,
Member2,
Member3,
// …
}

  • EnumName 是定义的枚举的名称。
  • Member1, Member2, Member3 等是枚举的成员，它们是枚举类型的常量值。

每个枚举成员都有一个默认值，从 0 开始，每个后续成员的值递增 1。也可以为枚举成员指定特定的整数值。

枚举一般声明在namespace语句块中，也可声明在class语句块和struct语句块中，但不能在函数语句块中声明。

声明枚举变量，需要先定义一个枚举类型，然后可以根据这个类型来声明变量。

声明枚举变量
DaysOfWeek today = DaysOfWeek.Monday;

如果枚举是嵌套的，需要使用类名来声明变量
Week.Day weekDay = Week.Day.Monday;

使用枚举变量
if (today == DaysOfWeek.Monday)
{
Console.WriteLine(“Today is Monday!”);
}

使用嵌套枚举变量
if (weekDay == Week.Day.Monday)
{
Console.WriteLine(“Today is Monday in the Week class context!”);
}

二维数组

声明一个具有指定行数和列数的二维数组：
int[,] arrayName = new int[rowCount, columnCount];

也可以在声明的同时初始化二维数组：

int[,] arrayName = new int[rowCount, columnCount]
{
{ element1, element2, …, elementN },
{ element1, element2, …, elementN },
…
};

交错数组

交错数组（Jagged Array）是一种特殊的二维数组，其中每一行可以有不同的列数。这意味着每一行都是一个单独的数组，因此每一行的长度可以不同。

在声明时初始化
int[][] jaggedArray = new int[][]
{
new int[] { 1, 2, 3 },
new int[] { 4, 5 },
new int[] { 6, 7, 8, 9 }
};

int[][] jaggedArray = new int[3][];

动态初始化每一行
jaggedArray[0] = new int[3] { 1, 2, 3 };
jaggedArray[1] = new int[2] { 4, 5 };
jaggedArray[2] = new int[4] { 6, 7, 8, 9 };

  • 交错数组的每一行都是一个独立的数组，因此它们可以有不同的长度。
  • 在访问交错数组的元素时，你需要确保不会超出每一行数组的界限，否则会抛出 IndexOutOfRangeException。
  • 交错数组在内存使用上可能不如规则的二维数组高效，因为它们包含额外的指针到每一行的数组。
